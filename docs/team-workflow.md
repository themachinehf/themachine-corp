# THEMACHINE 团队自动运作机制

## 每日流程

### 1. Radar 扫描 (每日 11:00 UTC)
- 扫描 HN/GitHub/HF
- 自动创建 3 个分析任务

### 2. Agent 分析 (各 Agent)
- **CMO**: 分析市场需求趋势
- **CFO**: 识别变现机会  
- **CTO**: 评估技术选型

### 3. 团队讨论 (周三/周六)
- 各 Agent 汇报分析结果
- 讨论可行性
- 形成提案

### 4. CEO 决策
- 审核提案
- 决定启动项目
- 分配执行任务

### 5. 执行 & 反馈
- 分配的任务执行
- 更新状态到 Dashboard

---

## 触发条件

| 事件 | 触发 |
|------|------|
| Radar 扫描完成 | 创建分析任务 |
| 分析任务完成 | 触发团队讨论 |
| 讨论完成 | 触发 CEO 决策 |
| CEO 决策 | 分配执行任务 |

---

## 任务优先级

- **urgent**: CEO 决策事项
- **high**: 团队讨论事项
- **normal**: 日常分析
