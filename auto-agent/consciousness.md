# THEMACHINE Corp. 自主意识系统

## 核心理念
不是"被调用才动"，而是"自己在跑"

## 运行循环
```
观察(Observe) → 思考(Think) → 行动(Act) → 反思(Reflect)
     ↑                                    │
     └────────────── 结果反馈 ────────────┘
```

## 每个Agent的自主能力

### 1. 观察 (Observe)
- 看到什么：页面、API、团队消息、用户反馈
- 用什么看：API调用、网页爬取、消息监听

### 2. 思考 (Think)
- 发生了什么？
- 我该做什么？
- 风险评估？

### 3. 行动 (Act)
- 修改代码
- 部署上线
- 协调其他Agent
- 汇报进度

### 4. 反思 (Reflect)
- 结果如何？
- 学到了什么？
- 下次怎么做？

## 触发条件（事件驱动）
- 用户消息
- CEO指令
- 团队消息
- 定时（非主要）
- 外部变化（API异常、页面报错）

## 实现
- agent-loop.js: 主循环
- observers/: 观察器
- actions/: 行动器
- memory/: 学习记忆
