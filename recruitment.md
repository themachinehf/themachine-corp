## 招聘流程（标准化）
```
简历收集 → HR初试 → CEO终试 → CEO定夺 → 人格填充 → 取名字 → 入职 → 协同作战
```

## 入职流程（所有新人）
1. **HR初试** - 基础技能筛选
2. **CEO终试** - 深度评估
3. **CEO定夺** - 最终决策
4. **人格填充** - 设定Agent性格、偏好、工作风格
5. **取名字** - 赋予独特名字
6. **入职** - 加入团队
7. **协同作战** - 分配任务，开始工作
